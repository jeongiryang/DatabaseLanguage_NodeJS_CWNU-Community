# v1.2.0 screenshot map review

## 매핑 기준

- 레포 루트에 새로 추가된 PNG/GIF만 `raw-v1.2.0` 원본으로 사용함.
- 기존 `docs/screenshots/raw/` 원본은 이번 문서 반영에 사용하지 않음.
- README 대표 영역은 GIF 중심으로 구성하고, 기능 설명서는 PNG 중심으로 갱신함.
- DB/Prisma Studio 관련 최신 캡처는 이번 신규 파일에 없어 기존 DB 캡처를 유지함.
- 루트 MP4는 이번 대상 확장자가 아니므로 이동/추적하지 않음.

## 확인 필요 항목

- GIF 6개는 단일 파일 100MB 미만이나 용량이 큰 편이므로 PR에서 파일 크기를 확인해야 함.
- 일부 GIF는 첫 프레임만으로 세부 동작을 완전히 판별하기 어려워 README에서는 대표 흐름 용도로 배치함.
- 모바일 PNG는 게시글 상세 중심 캡처이므로 모바일 전체 대시보드 흐름은 GIF 대표 파일을 함께 사용함.

## 사용하지 않은 최신 파일

- 없음. 모든 raw-v1.2.0 PNG/GIF는 하나 이상의 target에 매핑함.
